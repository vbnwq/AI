# 🎬 Free AI Studio — عکس‌ساز و فیلم‌ساز هوشمند (کاملاً رایگان)

یک هوش مصنوعی کامل برای ساخت **عکس** و **ویدیوی آموزشی** که در آن گوینده صحبت می‌کند —
بدون نیاز به هیچ **API key**، **توکن** یا **پرداخت پول**. از اول تا آخر رایگان.

شما فقط می‌نویسید چه می‌خواهید (مثلاً: «یک ویدیو درباره فرق لینوکس با ویندوز») و دکمه
ساختن را می‌زنید؛ برنامه خودش موضوع را تحلیل می‌کند، سناریو می‌نویسد، تصویر می‌سازد،
با صدای طبیعی صحبت می‌کند، زیرنویس می‌گذارد و یک ویدیوی کامل تحویل می‌دهد.

---

## ✨ امکانات

- **پشتیبانی کامل فارسی** + **لایه‌ی ترجمه خودکار (فارسی → انگلیسی)**: متن فارسی شما
  پیش از ارسال به مدل‌های تصویر، خودکار و دقیق به انگلیسی ترجمه می‌شود تا بهترین
  نتیجه ساخته شود (گوگل ترنسلیت رایگان + LLM + دیکشنری آفلاین داخلی).
- **عکس‌ساز قدرتمند و باکیفیت**: مدل رایگان **Flux** با تقویت‌کننده‌ی کیفیت
  (negative prompt + بوست جزئیات) و **شارپ‌سازی محلی** برای حداکثر وضوح و جزئیات.
  انتخاب کیفیت: استاندارد / بالا / **Ultra**.
- **فیلم‌ساز فارسی کامل**: ساخت ویدیوی آموزشی از یک جمله، با:
  - 🧠 نوشتن خودکار سناریوی **فارسی** (موتور آنلاین + نویسنده‌ی فارسی داخلی)
  - 🖼️ تصویر باکیفیت برای هر صحنه + انیمیشن نرم (Ken Burns)
  - 📝 **زیرنویس فارسی راست‌چین (RTL)** با فونت فارسی واقعی (Vazirmatn) که با PIL
    رندر می‌شود — حروف چسبیده، کاملاً خوانا، روی نوار شفاف زیبا
  - 🔊 صدای گوینده‌ی طبیعی نورال **فارسی** (زن/مرد) و هماهنگ‌شده با ویدیو
- **تاریخچه‌ی ساخته‌ها**: لیست همه‌ی عکس‌ها و ویدیوها با امکان **حذف تکی** یا **پاک‌کردن همه**.
- **دکمه‌های دانلود مجزا** برای عکس و ویدیو.
- **نمایشگر رسانه**: بزرگ‌نمایی/زوم عکس و نمایش تمام‌صفحه‌ی ویدیو (Lightbox).
- **انتخاب ابعاد و کیفیت**: 480p تا Full HD و عمودی استوری.
- **بهینه برای سخت‌افزار ضعیف**: بدون نیاز به کارت گرافیک؛ سه سطح کیفیت رندر
  (سریع / متعادل / حداکثر) تا روی CPU کم‌قدرت هم روان باشد.
- **`run.py`**: راه‌اندازِ خودکارِ همه‌چیز (نصب وابستگی‌ها + بررسی محیط + باز کردن مرورگر).
- **فایل اجرایی (EXE)**: با دابل‌کلیک باز می‌شود و مرورگر را خودکار باز می‌کند.

## 🆓 هوش مصنوعی‌های رایگان متصل‌شده (بدون API key)

| قابلیت | سرویس | توضیح |
|--------|-------|-------|
| تولید عکس | **Pollinations** (مدل Flux/Turbo) | رایگان، بدون کلید |
| نوشتن سناریو | **Pollinations Text** + موتور داخلی هوشمند | اگر آنلاین در دسترس نبود، خودکار آفلاین می‌نویسد |
| صدای گوینده | **Microsoft Edge TTS** + بکاپ Google TTS | صدای طبیعی نورال، چندزبانه |
| ساخت ویدیو | **FFmpeg** (محلی) | میکس تصویر + صدا + زیرنویس + انیمیشن |

> برنامه طوری ساخته شده که **هیچ‌وقت متوقف نشود**: اگر یک سرویس رایگان شلوغ بود،
> به‌صورت خودکار از منبع جایگزین یا موتور داخلی استفاده می‌کند.

## 🚀 روش استفاده (ساده‌ترین راه — همه‌جا)

فقط دستور زیر را اجرا کنید:

```bash
python run.py
```

فایل **`run.py`** همه‌چیز را خودکار انجام می‌دهد:
1. نسخه‌ی پایتون را بررسی می‌کند.
2. هر کتابخانه‌ی **نصب‌نشده** را خودش نصب می‌کند (فقط همان‌هایی که کم است).
3. وجود **FFmpeg** را بررسی می‌کند (روی ویندوز در صورت نبود، خودکار دانلود می‌کند).
4. پوشه‌های خروجی را می‌سازد.
5. سرور را بالا می‌آورد و **مرورگر را خودکار باز می‌کند**.

> دیگر نیازی به تنظیم دستی نیست — بعد از اجرا مستقیم شروع به استفاده کنید.

## 🚀 روش استفاده (ویندوز)

### روش ۱ — اجرای فوری (ساده‌ترین)
1. **Python 3.10+** را از [python.org](https://www.python.org/downloads/) نصب کنید
   (هنگام نصب تیک **Add Python to PATH** را بزنید).
2. روی فایل **`Start_FreeAIStudio.bat`** دابل‌کلیک کنید — یا در ترمینال `python run.py` را بزنید.
3. بار اول کمی صبر کنید تا بسته‌ها و FFmpeg نصب شوند؛ سپس مرورگر خودکار باز می‌شود.

### روش ۲ — ساخت فایل EXE مستقل
1. روی **`build_windows.bat`** دابل‌کلیک کنید.
2. صبر کنید تا ساخت تمام شود.
3. فایل ساخته‌شده در `dist\FreeAIStudio.exe` را هر جا ببرید و **دابل‌کلیک** کنید.
   (دیگر به نصب پایتون هم نیاز ندارید.)

## 🐧 اجرا روی لینوکس / مک

```bash
pip install -r requirements.txt
# FFmpeg باید نصب باشد: sudo apt install ffmpeg
python launcher.py
```
سپس مرورگر به آدرس `http://127.0.0.1:8000` باز می‌شود.

## 🧩 ساختار پروژه

```
webapp/
├─ run.py                 # ⭐ راه‌انداز خودکار همه‌چیز (نصب + مرورگر)
├─ launcher.py            # نقطه ورود جایگزین (دابل‌کلیک → مرورگر)
├─ FreeAIStudio.spec      # تنظیمات ساخت EXE با PyInstaller
├─ build_windows.bat      # ساخت EXE روی ویندوز
├─ Start_FreeAIStudio.bat # اجرای مستقیم روی ویندوز
├─ requirements.txt
├─ app/
│  ├─ server.py           # سرور FastAPI + Jobها + تاریخچه + دانلود
│  ├─ static/index.html   # رابط کاربری وب (تاریخچه، نمایشگر، دانلود مجزا)
│  ├─ fonts/              # فونت فارسی (Vazirmatn) و لاتین
│  └─ services/
│     ├─ translate.py     # 🆕 لایه‌ی ترجمه فارسی→انگلیسی
│     ├─ ai_text.py       # نوشتن سناریو (آنلاین + نویسنده‌ی فارسی داخلی)
│     ├─ ai_image.py      # تولید عکس باکیفیت (Flux + بوست + شارپ)
│     ├─ ai_tts.py        # تبدیل متن به گفتار فارسی نورال
│     └─ video_engine.py  # موتور ویدیو: FFmpeg + زیرنویس RTL با PIL
└─ output/                # عکس‌ها و ویدیوهای ساخته‌شده + تاریخچه
```

## ⚙️ نکات فنی

- ساخت تصاویر و صداها **به‌صورت موازی** انجام می‌شود تا ساخت ویدیو سریع باشد.
- اتصال کلیپ‌های صحنه با **stream-copy** انجام می‌شود (تقریباً آنی، بدون افت کیفیت).
- زیرنویس فارسی/عربی با `arabic-reshaper` و `python-bidi` اصلاح‌شکل و راست‌چین می‌شود.
- خروجی ویدیو: `H.264 + AAC` در `MP4` با `faststart` برای پخش روان.
